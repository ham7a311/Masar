# Masar 🚀

Masar is a student-focused platform designed to help university students discover learning resources, follow structured career roadmaps, and plan their learning journey in a more organized way.

It aims to reduce confusion caused by scattered information and provide a clear, guided path for skill development and career exploration.

---

## 🌐 Live Demo
👉 https://masar-17442.web.app/

---

## ✨ Features

- 📚 Curated learning resources by topic and skill
- 🧭 Structured career roadmaps
- 📊 Personal dashboard for tracking progress
- 🔐 Authentication (Google / GitHub / Microsoft)
- ⚡ Fast and responsive UI (built with Vite)
