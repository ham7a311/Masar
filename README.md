- Create and follow career roadmaps

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v16+ recommended) and npm or yarn
- A modern browser for the frontend

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/ham7a311/Masar.git
   cd Masar
   ```

2. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

### Running

Start the development server (adjust according to project scripts):

```bash
npm run dev
# or
npm start
```

Open http://localhost:3000 (or the port configured by the app) in your browser.
